# tokyo-mitsubishi-ufj-spot-rates
Meteor package - returns forex spot rates from Bank of Tokyo Mitsubishi UFJ.

## data source
http://www.bk.mufg.jp/gdocs/kinri/list_j/kinri/kawase.html

USD only for now.
